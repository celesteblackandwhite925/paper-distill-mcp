# Contributing

感谢你对 Paper Distill MCP 的关注！欢迎提交 Issue 和 Pull Request。

## 报告问题

- 使用 [Issue 模板](https://github.com/Eclipse-Cj/paper-distill-mcp/issues/new/choose) 提交 bug 或功能建议
- 提交 bug 时请附上：Python 版本、操作系统、错误日志

## 提交 PR

1. Fork 本仓库
2. 创建分支：`git checkout -b feat/your-feature`
3. 提交更改并推送
4. 创建 Pull Request，描述你的修改内容

### 开发环境

```bash
git clone https://github.com/<your-fork>/paper-distill-mcp.git
cd paper-distill-mcp
python3 -m venv .venv && source .venv/bin/activate
pip install -e .
```

### 代码规范

- 保持简洁，不要过度工程化
- 新功能请附带基本测试
- commit message 用英文，简明扼要

## 许可证

提交 PR 即表示你同意将代码以 [AGPL-3.0](LICENSE) 许可证发布。
